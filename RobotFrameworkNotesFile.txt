Topics - tags

How to set tags from cmd

Step 1 : Goto cmd
    CD to the location of your project folder

Step 2 : Run command
    robot -t TestName TestSuite

Step 3 : add flat to set tags
    --settag=tagname

Step 4 : Run tests with tags
    --include

Step 5 : Run tests with excluding tags
    --exclude
